How to use FileByte

Know how to share files between device between devices using FileByte

Connect the devices to the local network (Ethernet or Wifi)

Step One

  • Open FILEBYTE in your Device
  • On Home Page click Ready button
  • If you are connected to local network the code will generate
  • Wait for the connection

Step Two

  • Open FILEBYTE in another Device
  • On Home Page click connect button
  • Enter the code shown in other device
  • Wait for the connection

Step Three

  • The two devices will connect automatically
  • The sharing page will open in both devices ready for sharing

Step Four

  • Click the Share Button to select Files
  • The file selected will be shared

Connection Problem

If application are not connecting, try to change software port in both devices. To change port, go to settings then on port section click change and choose the port in selection bar. Note that the port must be the same in both devices.
